Privacy Policy

Last updated: May 2, 2026

This Privacy Policy describes how the Whoop Notion Sync app ("the App") accesses and uses your WHOOP data to synchronize selected health and workout metrics to your Notion workspace and a personal dashboard.

Data Collected

The App reads the following data from WHOOP APIs:

  • Recovery data (for example recovery score, HRV, resting heart rate)
  • Sleep data (for example sleep duration, sleep performance)
  • Cycle data (daily strain)
  • Workout data (for example sport type, strain, start/end times)

How Data Is Used

  • Workout records are synced to a user-owned Notion database.
  • Daily health metrics are written to a user-owned JSON file at data/data.json in this repository.
  • The JSON file is used to render a GitHub Pages dashboard for personal viewing.

Data Storage and Sharing

  • Data is stored only in services controlled by the user (GitHub repository and user-authorized Notion workspace).
  • The App does not sell personal data and does not share data with third parties beyond GitHub, Notion, and WHOOP as required for functionality.

Authentication and Tokens

The App uses WHOOP OAuth 2.0. Refresh tokens and API credentials are stored as GitHub Actions secrets (or local environment variables during setup) and are not intentionally exposed in public files.

Data Retention and Deletion

Data remains in your Notion database and repository until you delete it. You may stop synchronization at any time by removing app credentials/secrets or revoking WHOOP app access.

Your Choices

  • You can revoke WHOOP app authorization through your WHOOP account settings.
  • You can remove synced data from Notion and GitHub at any time.
  • You can disable the GitHub Actions workflow to stop future syncs.
